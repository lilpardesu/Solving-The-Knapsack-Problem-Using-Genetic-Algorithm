# 0/1 Knapsack Problem — Genetic Algorithm (PyGAD)

Solving the 0/1 Knapsack optimization problem using a Genetic Algorithm implemented with PyGAD.

## Problem Definition

Select items with given weights and values to maximize total value without exceeding knapsack capacity.

**Constraint:** Total weight ≤ Capacity  
**Encoding:** Binary chromosome (1 = selected, 0 = not selected)
